# سیستم تم‌های سازماندهی شده

سیستم تم‌های شما حالا به صورت مدولار و سازماندهی شده تنظیم شده است. هر تم در فایل جداگانه‌ای قرار دارد.

## ساختار فایل‌ها

```
themes/
├── index.css         # فایل اصلی که همه تم‌ها را import می‌کند
├── base.css          # تعاریف پایه (شامل @theme - اگرچه ممکن است warning داشته باشد)
├── zinc.css          # تم خاکستری سرد
├── slate.css         # تم آبی-خاکستری
├── stone.css         # تم خاکستری گرم
├── gray.css          # تم خاکستری خنثی
├── orange.css        # تم نارنجی
├── green.css         # تم سبز طبیعی
├── blue.css          # تم آبی اقیانوسی
├── yellow.css        # تم زرد روشن
├── violet.css        # تم بنفش
├── red.css           # تم قرمز پررنگ
├── black.css         # تم سیاه/سفید خالص
└── custom.css        # تم شخصی‌سازی شده (صورتی/گلابی)
```

## نحوه استفاده

### 1. Import کردن
در فایل اصلی theme.css فقط کافی است:
```css
@import './themes/index.css';
```

### 2. تغییر تم
برای تغییر تم از attribute `data-theme` استفاده کنید:
```html
<html data-theme="zinc">
<html data-theme="blue">
<html data-theme="green">
```

### 3. حالت تاریک
برای فعال کردن حالت تاریک، کلاس `dark` را اضافه کنید:
```html
<html data-theme="zinc" class="dark">
```

## تم‌های موجود

- **zinc**: خاکستری سرد و مدرن
- **slate**: آبی-خاکستری کلاسیک
- **stone**: خاکستری گرم و طبیعی
- **gray**: خاکستری خنثی
- **orange**: نارنجی پر انرژی
- **green**: سبز طبیعی و آرامش‌بخش
- **blue**: آبی اقیانوسی (پیش‌فرض)
- **yellow**: زرد روشن و شاد
- **violet**: بنفش خلاقانه
- **red**: قرمز قدرتمند
- **black**: سیاه/سفید کنتراست بالا
- **custom**: صورتی/گلابی شخصی‌سازی شده

## ویژگی‌ها

### متغیرهای یکپارچه
همه متغیرها با پیشوند `--color-` شروع می‌شوند:
- `--color-primary`
- `--color-background`
- `--color-foreground`
- و غیره...

### سازگاری با قبل
متغیرهای alias برای سازگاری با کدهای قبلی:
- `--primary` → `var(--color-primary)`
- `--background` → `var(--color-background)`

### پالت رنگ کامل
هر تم شامل پالت رنگ کامل 50-950 برای رنگ اصلی

### Tonal Variants
برای هر تم، متغیرهای tonal برای استفاده در کامپوننت‌ها

## اضافه کردن تم جدید

1. فایل جدید در پوشه themes/ ایجاد کنید
2. ساختار مشابه تم‌های موجود استفاده کنید
3. آن را در index.css import کنید
4. هر دو حالت light و dark را تعریف کنید

## نکات مهم

- همه فایل‌ها از CSS خالص استفاده می‌کنند
- سیستم از HSL برای تعریف رنگ‌ها استفاده می‌کند
- حالت تاریک برای همه تم‌ها پشتیبانی می‌شود
- سیستم با فونت‌های فارسی سازگار است
